LEVEL 1. 클릭 유발하는 썸네일 제작

고퀄리티 이미지 생성 프롬프트로 홍보 이미지, 카드뉴스, 교육 안내 배너를 더 눈에 띄게 제작하는 방법입니다.
LEVEL 1 이미지 생성 초급 활용
이런 상황에 추천
  • 홍보용 이미지가 밋밋해서 클릭률을 높이고 싶을 때
  • 행사·특강·모집 안내를 한 장 이미지로 빠르게 만들어야 할 때
  • 디자인 시간을 줄이면서도 시각적 임팩트를 확보하고 싶을 때

클릭을 부르는 구성 원리

  • 핵심 메시지는 1개만 잡고 크게 강조하기
  • 대비가 큰 색상 조합으로 시선 집중 유도하기
  • 텍스트는 후편집을 전제로 두고 이미지에는 여백 확보하기
  • 대상에 따라 분위기를 명확히 지정하기

이미지 생성 프롬프트

다음 조건에 맞는 홍보용 썸네일 이미지를 제작하기 위한 프롬프트를 작성해 주세요.

[목적] 대학 내 프로그램/행사 홍보
[대상] 학생 및 교원
[핵심 메시지] 한눈에 들어오고 클릭하고 싶게
[분위기] 전문적이지만 젊고 세련된 느낌
[색감] 대비가 분명하고 밝은 포인트 컬러 사용
[구성] 중앙 핵심 오브젝트 + 여백 확보 + 제목 후편집 가능 구조
[스타일] 디지털 교육, AI, 미래지향, 신뢰감
[금지 요소] 과도한 텍스트 삽입, 복잡한 배경, 산만한 오브젝트

출력은 다음 형식으로 작성해 주세요.
1. 이미지 생성용 최종 프롬프트
2. 짧은 버전 프롬프트
3. 더 자극적으로 보이게 개선한 버전

LEVEL 2. 손필기 → PPT/문서 변환

메모를 구조화된 초안으로 전환해 회의 메모, 보드판 필기, 러프한 아이디어를 문서 또는 발표자료로 정리하는 방법입니다.
활용 흐름
  1. 손필기 또는 보드판을 선명하게 촬영
  2. 누락 없이 텍스트를 읽게 한 뒤
  3. 문서 목적에 맞는 구조로 재정리 요청
  4. 최종 초안 검토 후 제목·표현 보정

문서용 프롬프트

내가 업로드한 손필기/메모 이미지를 읽고, 내용을 누락 없이 정리해 주세요.

조건:
- 의미가 불명확한 부분은 임의로 확정하지 말고 [확인 필요]로 표시
- 중복 표현은 정리하되 원래 취지는 유지
- 결과는 실무 문서 초안 형태로 재구성
- 아래 순서로 출력

1. 핵심 주제
2. 주요 내용 정리
3. 실행 항목
4. 추가 확인이 필요한 부분
5. 바로 문서에 붙여넣을 수 있는 정리본

문장은 너무 장황하지 않게, 행정 실무자가 바로 수정 가능한 수준으로 작성해 주세요.
PPT용 전환 포인트
  • 문장보다 목차·키워드 중심으로 재구성
  • 슬라이드 1장당 메시지 1개 원칙 적용
  • 서론-핵심-결론 구조로 묶어 활용성 높이기

PPT용 프롬프트

내가 업로드한 손필기/메모 이미지를 읽고 PPT 초안용으로 재구성해 주세요.

조건:
- 누락 없이 핵심 내용을 정리
- 의미가 불명확한 부분은 [확인 필요] 표시
- 슬라이드 목차 구조로 변환
- 각 슬라이드는 제목 + 핵심 bullet 3~5개 이내로 작성
- 마지막에는 전체 발표 흐름 요약 추가

출력 형식:
1. 전체 발표 제목 제안 3개
2. 슬라이드 목차
3. 슬라이드별 핵심 bullet
4. 발표 시 보완이 필요한 부분
회의록화 핵심
  • 참고 메모를 의사결정, 논의사항, 후속조치로 분리
  • 날짜·담당·기한 등 빠진 정보는 확인 필요 표시
  • 실행 항목 중심으로 정리하면 바로 활용 가능

회의록용 프롬프트

첨부한 손필기 메모를 읽고 회의록 형태로 정리해 주세요.

조건:
- 확실한 정보만 작성하고 불명확한 내용은 [확인 필요] 표시
- 아래 구조로 출력

1. 회의 주제
2. 주요 논의사항
3. 결정사항
4. 후속 조치 항목(담당/기한 포함, 없으면 빈칸 표시)
5. 원문 메모 기준 누락 가능 지점

출력은 간결하고 실무적인 문장으로 작성해 주세요.

LEVEL 3. 답변 품질 UP LLM 스킬

근거·형식·품질 제어 프롬프트를 활용해 더 정확하고 안정적인 답변을 얻는 방법입니다.
LEVEL 3 Skill 01

환각 줄이기

모르면 모른다고 답하게 하고, 불확실한 내용은 분리 표시하게 만드는 방식입니다.

확실한 정보만 답변하고, 불확실한 내용은 추정하지 마세요.
근거가 부족한 경우에는 [확인 필요]라고 표시하세요.
답변은 사실 / 추정 / 추가 확인 필요 항목으로 구분해 주세요.
LEVEL 3 Skill 02

근거 범위 설정

엉뚱한 일반론을 줄이려면 자료 범위와 출처 범위를 먼저 고정해야 합니다.

답변 범위는 내가 제공한 자료와 공식 기준으로 한정해 주세요.
범위를 벗어나는 일반론은 최소화해 주세요.
필요하면 어떤 자료가 더 있어야 정확도가 올라가는지도 함께 알려주세요.
LEVEL 3 Skill 03

답변 형식 고정

불필요한 장문을 줄이고, 필요한 결과만 빠르게 받아보는 데 효과적입니다.

답변은 아래 형식으로만 작성해 주세요.
1. 핵심 결론 3줄
2. 실행 항목 5개
3. 주의사항 2개
불필요한 설명은 제외해 주세요.

LEVEL 4. 이번 달 신규 LLM 모델

업데이트 요약과 추천 용도를 확인하고, 작업 성격에 맞는 모델을 선택하는 방법입니다.
LEVEL 4 빠른 응답형

Model A

짧고 빠른 응답에 강하며, 일상적인 질의응답과 간단한 초안 생성에 적합한 모델입니다.

추천 용도: 짧은 요약, 안내문 초안, 빠른 질의응답

LEVEL 4 문서 정리형

Model B

긴 문서 요약과 구조화에 강점을 보이며, 회의자료·보고서·기획안 정리에 적합한 모델입니다.

추천 용도: 장문 요약, 회의록 정리, 문서 초안 재구성

LEVEL 4 아이디어 확장형

Model C

홍보 문구, 제목 제안, 아이디어 확장처럼 창의적 초안 생성에 활용하기 좋은 모델입니다.

추천 용도: 홍보 카피, 행사명 제안, 시안 초안, 브레인스토밍

모델명 핵심 특징 추천 사용 사례 활용 팁
Model A 빠른 응답, 가벼운 작업에 적합 짧은 요약, 간단한 안내문 초안, 빠른 질의응답 복잡한 분석보다 빠른 초안 확인용으로 활용
Model B 긴 문서 구조화와 요약에 강점 회의록, 기획안 초안, 보고서 정리 자료 범위와 출력 형식을 함께 지정하면 안정적
Model C 창의적 표현과 아이디어 확장에 유리 홍보 문구, 썸네일 카피, 행사명 제안 사실 확인이 필요한 업무에는 최종 검토 필요

빠른 복사용 프롬프트 모음

각 레벨에서 소개한 핵심 프롬프트만 모아, 필요한 순간 바로 복사해 사용할 수 있도록 정리했습니다.
LEVEL 1

썸네일 제작 프롬프트

대학 프로그램 홍보용 썸네일 이미지를 만들고 싶습니다. 학생 대상, 밝고 세련된 분위기, 핵심 메시지가 한눈에 들어오게 구성해 주세요. 제목 삽입을 위한 여백을 확보하고, 복잡한 배경은 피해주세요.
LEVEL 2

손필기 문서화 프롬프트

첨부한 손필기 내용을 누락 없이 읽고 실무 문서 초안으로 정리해 주세요. 불명확한 부분은 [확인 필요]로 표시하고, 핵심 내용 / 실행 항목 / 추가 확인 사항으로 구분해 주세요.
LEVEL 3

환각 감소 프롬프트

확실한 정보만 답변하고, 불확실한 내용은 추정하지 마세요. 근거가 부족한 부분은 [확인 필요]라고 표시하고, 사실과 추정을 구분해 주세요.
LEVEL 3

답변 형식 고정 프롬프트

답변은 아래 형식으로만 작성해 주세요. 1) 핵심 결론 3줄 2) 실행 항목 5개 3) 주의사항 2개. 불필요한 설명은 제외해 주세요.
